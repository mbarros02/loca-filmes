<h1>LocaFilmes - Locação e Compra de Filmes</h1>
<h2>Um sistema de locação e compras de filmes.</h2>

<p>O sistema LocaFilme é um projeto do curso de ADS - Análise e Desenvolvimento de Sistemas, na instituição SENAC - SP. 
  Esse sistema foi criado para ser funcional via console e para a prática de Design Patterns, conteúdo aprendido na matéria de Arquitetura de software do 4º semestre do curso.</p>
